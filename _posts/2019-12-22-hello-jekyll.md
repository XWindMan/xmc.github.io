> 本站使用方法

+ 内容
  ```
  在_post路径下，编写内容
  编写头
  ---
  layout: post
  title: 'Hello Jekyll'
  date: 2019-12-22
  author: WindMan
  cover:'http://a3.att.hudong.com/14/75/01300000164186121366756803686.jpg'
  tags: Tools
  ---
  ```
+ 编译
``` 
jekyll build
```
+ 浏览器预览
``` 
jekyll server
// url
Server address: http://127.0.0.1:4000/
```

+ 图片相对路径引用
```
/assets/*
```

+ 字体切换
```
/assets/font
```

+ 提交即可
+ [Jekyll主题](http://jekyllthemes.org/)
