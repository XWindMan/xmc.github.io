+ 支持和JAVA混合开发

+ 变量声明
```
+ 变量声明比JAVA多了个var（变量）、val(常量)，其他类型会自动推断
```

+ 函数
```
+ 函数声明用fun
+ override 覆盖超类方法,修饰fun
```

+ 类
```
+ class声明类，没有new
+ 类的继承，用：，超类在后
+ 所有对象的超类是 Any
+ abstract修饰class
+ 主构造函数和类声明在一起，次级构造函数可以通过this委托

+ 内部类用inner修饰

+ 只保存数据的类用data修饰
+ 泛型同java
+ 接口用enum修饰
```
